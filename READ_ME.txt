2. 페이지 구성
- 메인페이지
   기능1 : 스크롤 시 헤더와 닿으면 픽스드 되며 따라옴 색상 반투명
   기능2 : 헤더 메뉴에 호버하면 100% 서브메뉴판 나옴  
   기능3 : 서브메뉴 판에서 두꺼운 글씨 누르면 product로 이동(아이콘 눌러도 이동)
   기능4 : 메인배너 auto slide, hover시 멈춤, 인디케이터 눌러도 이동
   기능5 : 섹션 마다 중간에 올때쯤 opacity1로 바뀌며 이동함.
   기능6 : celeb pick에 호버하면 설명 올라옴, 버튼 누르면 해당 item 페이지로 이동
   기능7 : 베스트 섹션 옆 카테고리 누르면 바뀜, 아이템 누르면 item 페이지로 이동
   기능8 : MD픽 아이템 호버시 사진 바뀜, 아이템 누르면 item페이지로 이동
   기능9 : 베스트 리뷰 스와이퍼
   => 메인페이지 반응형 이므로 최소 사이즈 760까지 줄일 수 있음
   기능10 : 760사이즈가 되면 헤더가 바뀌며 햄버거 버튼이 나타남
   기능11 : 햄버거 버튼을 누르면 왼쪽에서 슬라이드 메뉴가 들어옴
   기능12 : 아코디언 메뉴 중 뷰티 메뉴 안에 글씨 클릭 시 해당 product로 이동

- 프로덕트 페이지
   기능1 : 처음 페이지에 들어오면 아무것도 없지만 스크롤 하면 리스트 갯수만큼 나타남
   기능2 : 아이템 호버시 장바구니 및 like 아이콘 나타나며 서브 사진 나옴
   기능3 : 아이템 누르면 해당 item 사이트로 이동함
   => 반응형이므로 최소 사이즈 760까지 줄일 수 있음

- 아이템 페이지
   기능1 : 사진에 호버하면 확대경 사진 볼수 있음
   기능2 : select에서 클릭하면 해당 아이템 들어감 -> 같은 아이템을 두번 넣으려고 하면 경고창 뜸
   기능3 : 수량을 올리거나 내릴 수 있고 1미만으로 숫자를 조정하려 할 경우 경고창 뜸
   기능4 : 넣은 아이템을 뺄 수 있고 담긴 아이템의 갯수에 따라 가격이 바뀜
   기능5 : 아래 디테일 메뉴를 누르면 해당 섹션으로 이동함
